# Coswara_Covid19_detection
This project studies the possibility of detecting covid19 from respiratory acoustics like cough, breathing and speech sounds. It uses public Coswara dataset (https://github.com/iiscleap/Coswara-Data) for the study. This was an elementary study. Detailed study and associated codes can be found at: https://github.com/iiscleap/codes-in-tool-release .
